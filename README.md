# 微信公众号 DeepSeek AI 机器人

一个运行在腾讯云函数上的微信公众号 AI 对话机器人。

## 部署步骤

### 1. 登录腾讯云
打开 [腾讯云控制台](https://console.cloud.tencent.com) 登录

### 2. 创建云函数
1. 进入「云函数」服务
2. 点击「新建」
3. 选择「从头开始」
4. 函数名称：`wechat-ai-bot`
5. 运行环境：Node.js 18.15
6. 点击「下一步」→「完成」

### 3. 上传代码
1. 进入刚创建的函数
2. 点击「函数代码」标签
3. 把 `index.js` 的内容复制粘贴到编辑器
4. 点击「保存」

### 4. 创建触发器
1. 点击「触发管理」标签
2. 点击「创建触发器」
3. 触发方式：API网关
4. 勾选「启用集成响应」
5. 点击「提交」
6. 复制生成的访问路径（URL）

### 5. 配置微信公众号
1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入「设置与开发」→「基本配置」
3. 修改服务器配置：
   - **URL**: 填第4步复制的 API 网关地址
   - **Token**: `myai2024`
   - **EncodingAESKey**: 点击「随机生成」
4. 点击「提交」

### 6. 启用并测试
- 提交成功后，点击「启用」
- 手机打开公众号，发消息测试！

---

## 配置参数说明

代码中的配置已经写好了：
- `DEEPSEEK_API_KEY`: DeepSeek API Key
- `APP_ID`: 微信公众号 AppID
- `APP_SECRET`: 微信公众号 AppSecret
- `TOKEN`: 自定义 Token（配置公众号时用）

## 常见问题

**Q: 提示 URL 超时？**
A: 腾讯云函数首次调用可能较慢，等几秒再试。

**Q: 返回"请求超时"？**
A: 检查 API Key 是否正确，或 DeepSeek 服务是否正常。

**Q: 如何修改 AI 回复风格？**
A: 在 `index.js` 中找到 `callDeepSeek` 函数，修改 system prompt 内容。
